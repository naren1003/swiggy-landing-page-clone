# Swiggy Landing Page Clone

This project is a responsive landing page inspired by the Swiggy homepage.  
Built using HTML and CSS to practice layout, responsiveness, and modern UI design.

##  Features
- Responsive navigation bar
- Hero section with call-to-action
- Service highlights section
- App download section
- Footer with links
- Mobile responsive design

##  Tech Stack
- HTML5
- CSS3 (Flexbox & Grid)

##  Responsive Design
The page is fully responsive and works on:
- Mobile devices
- Tablets
- Desktop screens

##  Purpose
This project was built to improve frontend development skills by recreating a real-world landing page layout.
